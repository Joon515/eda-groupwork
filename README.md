# EDA Groupwork Repository

This repository contains files and resources for an EDA coursework project based on FIFO design and verification.

## Repository Overview

```txt
eda-groupwork
├── design_src              // repository for design files
│   ├── old_design2v        // obsolete reference design
│   ├── origin_design2sv    // original SystemVerilog design
│   └── recent_design2sv    // latest SystemVerilog design
├── icarus                  // Icarus Verilog simulation files
└── netlist                 // generated netlist
```

## Notes

+ This repository is intended for coursework development and includes both older and latest design versions for comparison and iteration.
+ The project structure is not yet complete. If you want to start quickly, copy `design_src/recent_design2sv/`.
+ If you are interested, fork this repository and submit your pull request.
